Описание реализации
Этот сельскохозяйственный симулятор реализован с использованием объектно-ориентированного программирования в JavaScript:

Основные классы:
Cell (Клетка) - базовый класс для клеток игрового поля:

Имеет тип (земля/вода) и уровень влажности

Меняет цвет в зависимости от влажности

Может содержать растение

Plant (Растение) - базовый класс для растений:

Определяет минимальную и максимальную влажность для роста

Имеет уровень роста и скорость роста

Погибает при неподходящей влажности

Potato, Cactus, SwampPlant - классы конкретных растений:

Наследуют от Plant

Определяют уникальные параметры для каждого растения

Game - основной класс игры:

Управляет игровой логикой

Обрабатывает взаимодействие с пользователем

Обновляет состояние игры

Функциональность:
Инструменты: лопата, ведро с водой, пустое ведро

Три типа растений с разными требованиями к влажности

Динамическое изменение влажности и цвета земли

Визуализация роста растений

Информационная панель о состоянии клетки

Система полностью соответствует требованиям задания и демонстрирует использование ООП-парадигмы в JavaScript.
